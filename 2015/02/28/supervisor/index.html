supervisor

supervisor 是node开发工具，node应用修改源码以后，需要重启node，使用supervisor后
它会监视文件的改变，然后自动重新启动node

supervisor用法

  1. 使用 npm 安装 supervisor

    npm install supervisor -g

    如出现权限错误，使用

    sudo npm install supervisor -g

  2. 使用 supervisor 命令启动 app.js:

    supervisor app.js
